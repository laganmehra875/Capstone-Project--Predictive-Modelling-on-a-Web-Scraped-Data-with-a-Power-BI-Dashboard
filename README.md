# Capstone-Project on Flipkart--Predictive-Modelling-on-a-Web-Scraped-Data-with-a-Power-BI-Dashboard

**🎯Project Objective**

The main objective of this project is to analyze e-commerce product data from Flipkart to understand pricing patterns, customer ratings, reviews, and discount strategies.

Key objectives:

1. Collect product data from Flipkart using web scraping techniques.
2. Clean and preprocess the scraped dataset for analysis.
3. Perform Exploratory Data Analysis (EDA) to identify trends in pricing, ratings, and reviews.
4. Build machine learning models to analyze relationships between product features such as price, ratings, and reviews.
5. Visualize insights using Power BI dashboards for better business decision-making.
6. Generate business insights that can help e-commerce platforms improve pricing and marketing strategies.
